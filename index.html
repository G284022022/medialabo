本テーマの学習の目的

このテーマでは，HTML, CSS, JavaScript の基本を学び，これらを組み合わせて Web ページを構成する方法を学びます． HTML によって Web ページの構造を記述する基本的な方法、 CSS によって Web ページの見た目を設定する基本的な方法、 JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる。 HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる。

プロジェクト課題について

プロジェクト課題は番組表、グルメ情報、世界各地の天気の中から好きのものを一つ選んで各自で情報を検索することができるwebページを作成した課題だ。プロジェクト課題で、私のプロジェクト課題は世界の天気を選択した。世界の天気が表示されるwebページを作りました。 都市名の名前を選択して検索をすると、都市名、最高気温、最低気温、湿度が表示されるようになっている。

    第1回 WWWとHTML

    今回はWebページとWWWの特徴、WWWとその技術要素、 HTMLテキストの要素、ハイパーリンク、画像の埋め込みを勉強しました、 また次の課題を取り組みました、HTML文書の作成、em要素、属性の追加、 HTMLのさまざまな要素。さまざまな大きさの見出しの作り方や、リスト、説明や、URLの付け方、強調などを学んだ。

    • 演習課題:HTML文書の作成
    • 演習課題: HTMLのさまざまな要素

    第2回 CSSによるWebページの装飾

    前回学んだ HTML は文書の構造とコンテンツを記述するものでした． また，HTML は要素の集まりでできています． この要素それぞれの見た目の設定（装飾）について今回学びまました．文字のフォント、URLなどの色や、デザイン、などを設定した。他にも、影や、グラデーションをつけた。また、下線を引いたり、グリッドを使ったりした

    • 演習課題:ウェブブラウザ説明ページの装飾

    第3回 JavaScriptの基本

    今回のJavaScriptはWeb ページ上で複雑な機能を実現するプログラミング言語です． Web ページを開いた直後や，ボタンを押したときに JavaScript プログラムが実行されます．ページの自動更新やアニメーションなど動きのあるページを実現するために JavaScript プログラムを使います．オブジェクトやメンバー、javaScriptにおけるfor文について学び、演習をおこなった。オブジェクトは{..}でメンバーを指定し、データを項目ごとにまとめる。また、[]を用いて配列を作り、データを使用しやすくする。

    • 演習課題:オブジェクトの参照

    第4回 DOMとJavaScript

    DOM（Document Object Model）は Web コンテンツを表現するデータ構造です．DOM を使用することによって，JavaScript などのプログラムによって，Web コンテンツの内容を読み取ったり，コンテンツに変更を加えたりする処理が書けるようになります． 言い換えると，DOM と JavaScript を組み合わせることによって，動きのある Web ページを実現できるようになります．

    • 演習課題：DOMプロブラミングとJavaScript

    第5回 イベント処理

    今回はイベントが発生したときに JavaScript プログラムが動作するような処理「イベント処理」の実現方法を学びます．button要素とinput要素ついて学び、その中で特に、button要素でイベント処理、inputでlabelやtypeなどを考えた。イベント処理では、どのように発火するか設定でき、クリックや、フォーカスなどを使った

    • 演習課題:イベントを選択する

    私の工夫点，苦心点，強調点，感想

    情報メディア実験の授業を通じて、簡単なホームページの作り方を学んで、本当に嬉しかった。まずは西田先生と2人の先輩に感謝したいと思います。西田先生は授業中に複雑な知識の内容を丁寧に授業し、説明してくれたので、難しいものを理解しやすくなりました。2人の先輩も私がわからない時にヒントやアドバイスをくれて、本当にありがとうございました。

    私は中国人留学生として、私の日本語のレベルは日常会話では問題ありませんが、この授業を始めたばかりの頃は理解しにくいカタカナ日本語がたくさんありました。それは私にとって本当に挑戦でした。その時私は最初から各種ソフトウェアのインストールがあまりできなくて、それから後になって各種フォルダを作成して、更にhtmlの原理、構成などの各種知識を理解して、更にcssの知識など、私は最初から今まで簡単なホームページを作ることができて、この間は多くの困難な挑戦を経験して、本当に大変でした。

    私はWebの学習を通じて、多くの感想を抱いています。まず、Webの学習は非常に魅力的です。HTMLやCSS、JavaScriptなどの基本的な技術を学び、それらを組み合わせてウェブページを作成することができます。最初は難しく感じるかもしれませんが、実際に手を動かしてコードを書くことで、少しずつスキルが向上していくのが実感できます。自分の手で作り出す喜びや達成感を味わえるのは、とても素晴らしいことです。また、Webの学習は自己表現の手段としても非常に魅力的です。デザインやレイアウト、色彩の組み合わせなど、自分のアイデアやセンスをウェブページに反映することができます。

    課題の中で最も苦労したものが数あてゲームを作成する課題だ。この課題はランダムに設定された数字をユーザーが実際に検索欄に記入して３回以内に当てられたらクリアというゲームだが、このゲームを作成することが非常に難しかった。

    Webの学習は将来のキャリアにも繋がると感じています。Web開発のスキルを持つことは、求人市場で競争力を持つことに繋がります。また、自分自身でウェブサイトやアプリケーションを作成することも可能です。ただし、Webの学習には挑戦も伴います。しかし、この挑戦も成長の機会でもあります。継続して努力を重ねることで、自身のスキルを向上させることができます。

    総じて、Webの学習は楽しく充実感のあるものです。創造性を発揮し、自己表現をすることができる上に、将来のキャリアにも繋がると感じます。挑戦もあるかもしれませんが、それを乗り越えることで成長できるのは素晴らしい経験です。これからもWebの学習を続け、さらにスキルを磨いていきたいと思います。

    最後にもう一度西田先生と2人の先輩がこれまで私たちの学生に関心を持ってくれたことに感謝したいと思います。本当にありがとうございました。
